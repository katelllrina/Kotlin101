# Kotlin Syntax

## Operators

### Arithmetics  

- `+`: addition/plus
- `-`: subtraction/minus
- `*`: multiplication
- `/`: division/div
- `%`: modulus/mod

### Comparison

- `<`: less
- `>`: greater
- `<=`: less or equal
- `>=`: greater or equal
- `==`: equal
- `!=`: not equal

## Keywords

- `fun`: define new function
  - `fun sumDigit(n: Int): Int { ... }`
- `val`: create new immutable variable
- `var`: create new   mutable variable
- `if-else`: condition
  - `if (x < 10) { ... } else { ... }`
  - `if (x < 10) { ... }`
- `return`: return result from function and end function
  - `return n`
  - `return 10`
  - `return n + 7`
  - `return lastDigit(n) * 10`
- `while`: conditional cycle
  - `while (n < 10) { n += 1 }`
