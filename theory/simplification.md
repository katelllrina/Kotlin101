# Simplification rules

## Arithmetics

| **Before**  | **After** | **When** |
|-------------|-----------|----------|
| `n = n + x` | `n += x`  | Always   | 
| `n = n - x` | `n -= x`  | Always   | 
| `n = n * x` | `n *= x`  | Always   | 
| `n = n / x` | `n /= x`  | Always   | 
| `n = n % x` | `n %= x`  | Always   |
